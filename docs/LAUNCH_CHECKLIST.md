## 2. Security Hardening

- [x] **Rate limit** — login 10/min, global 200/min (slowapi)
- [x] **Security headers** — X-Frame-Options, X-Content-Type-Options, Referrer-Policy, Permissions-Policy
- [x] **HSTS** — auto-enabled when serving over HTTPS
- [x] **Password policy** — minimum 8 chars enforced backend + frontend
- [x] **JWT** — bearer token in Authorization header (no cookie/CSRF risk)
- [x] **SQL injection safe** — SQLAlchemy parameterized queries everywhere
- [x] **XSS safe** — React JSX auto-escape; no `dangerouslySetInnerHTML`
- [x] **Audit log** — `audit_log` table records: login, user.create, user.delete, price.override, etc.
- [ ] **HTTPS cert** — pastikan nginx serve `https://dapurpintarmbg.aureonforge.com` dengan Let's Encrypt cert valid (manual)
- [ ] **Strong SECRET_KEY** — minimal 32 chars random di production `.env` (manual)
- [ ] **Rotate scanner_key + cloud_print_key** kalau pernah di-share di chat/email (manual via UI)
